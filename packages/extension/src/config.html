BazaarInfo Companion Setup

The companion app watches your game and sends card positions to the overlay so viewers can hover over cards for tooltips.

Authenticating with Twitch…

Quick Start

  1. Download the companion from the latest release
  2. Run it — it will ask for your Channel ID and Secret shown above
  3. Launch The Bazaar and start a game
  4. Viewers in fullscreen will see card tooltips on hover

Manual Setup (Python)

  1. Clone the repo and go to packages/companion/
  2. Install deps: pip install -r requirements.txt
  3. Edit config.ini with your Channel ID and Secret
  4. Run: python logwatch.py
